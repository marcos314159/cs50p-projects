# CS50P Exercises

Solutions developed during Harvard University's CS50P (Introduction to Programming with Python).

## About

This repository contains my solutions to the programming problems completed throughout the CS50P course.

The exercises cover a variety of Python concepts, including:

- Variables and conditionals
- Loops
- Functions
- Exceptions
- File I/O
- Regular expressions
- Object-oriented programming
- Unit testing with pytest

## Repository Structure

Each folder contains the solution for a specific problem from the course.

Examples include:

- adieu
- bank
- bitcoin
- fuel
- grocery
- professor
- watch

## Technologies Used

- Python
- pytest
- Regular Expressions (re)
- CSV
- File Handling
